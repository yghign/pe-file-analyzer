# PE文件分析器

一个功能全面的PE（Portable Executable）文件分析工具，提供图形化界面。

## 功能特性

### 1. 文件头解析
- DOS头、NT头、可选头分析
- 架构识别（32位/64位）
- 数据目录表解析

### 2. 节表分析
- 详细节表信息显示
- 权限分析（可读、可写、可执行）
- 熵值计算（检测加壳）
- 安全风险评估

### 3. 安全分析
- 哈希值计算
- 字符串提取与分类
- 导入函数安全检测
- 可疑行为识别

### 4. 图形界面
- 直观的选项卡导航
- 树形视图展示
- 综合报告生成
- 一键导出功能

## 安装使用

### 方法一：使用打包版本（推荐）
1. 下载最新的发布版本
2. 解压缩文件
3. 找到dist文件夹
4. 双击 `PEanalyze.exe` 运行

### 方法二：从源码运行
```bash
# 克隆项目
git clone https://github.com/你的用户名/PE-File-Analyzer.git

# 安装依赖
pip install -r requirements.txt

# 运行程序
python PEanalyze.py
